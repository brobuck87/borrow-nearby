# Borrow Nearby

A site dedicated to helping users find and market their tools and goods for nearby neighbors to use

## Technology

- HTML
- CSS
- Javascript
- React
- Express
- Node
- Sequelize
- Handlebars
- MySQL
- 3rd Party API's

## User Stories

- As a user, I can post a tool or a good for someone to find near them
- As a user, I can find a tool or good near me
- As a user, I can log in to my account
- As a user, I can see the cost of renting the tool or good versus buying it
- As a user, I can see a map with my nearby posters and their tools
